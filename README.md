# Homethings

A personal homelab of mini apps. Made up of a static personal website & a web App built using Pocketbase & React.

## WEB APP ENVIRONMENT VARIABLES

- ENV
- STORAGE_KEY
- STORAGE_SECRET
- STORAGE_ENDPOINT
- STORAGE_BUCKET
- STORAGE_BACKUP_BUCKET
- SMTP_SERVER
- SMTP_PORT
- SMTP_USERNAME
- SMTP_PASSWORD
- OPENROUTER_API_KEY
